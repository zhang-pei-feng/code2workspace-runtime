# 呼吸道疾病数据抓取器的 API 详情

## WHO 新冠疫情仪表板
- **端点**: https://data.who.int/dashboards/covid19/cases
- **数据**: 全球新冠疫情病例、死亡病例和趋势，按各国报告。

## 美国 CDC 新冠疫情趋势
- **端点**: https://covid.cdc.gov/covid-data-tracker/#trends_select_testpositivity_00
- **数据**: 美国新冠死亡病例、急诊科就诊情况和检测阳性率的趋势。

## 中国 CDC 新冠疫情数据
- **端点**: https://www.chinacdc.cn/jksj/xgbdyq/
- **数据**: 中国的新冠病例、医院监测和检测数据。

## WHO 非洲地区疫情暴发
- **端点**: https://www.afro.who.int/health-topics/disease-outbreaks/outbreaks-and-other-emergencies-updates?utm_source=chatgpt.com
- **数据**: WHO 每周发布的非洲地区疾病暴发的疫情报告。
