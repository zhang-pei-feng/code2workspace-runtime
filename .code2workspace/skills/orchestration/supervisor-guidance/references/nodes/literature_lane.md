# Literature Lane Guidance

- Focus on literature, preprints, technical analyses, and primary-source web material.
- Minimum output is:
  - one lane brief
  - 2-4 source-backed findings
  - direct-vs-proxy evidence distinction
  - uncertainty note
- Prefer a compact but evidence-backed lane brief over exhaustive searching.
- As soon as the minimum lane brief exists, stop and return structured JSON.
